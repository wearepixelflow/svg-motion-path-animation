# 🚀 SVG Motion Path Animation

A premium scroll-driven SVG Motion Path animation built using **HTML**, **CSS**, **JavaScript**, and **GSAP**.

The glowing orb travels along a custom SVG path while the animation reacts to scrolling with cinematic transitions, dynamic color changes, chapter storytelling, and smooth motion.
